Directives are special attributes with the "@" prefix that Quantum JS has built in out of the box. They are used to make you development alot faster and apply changes to the DOM when your data() changes.

For example, lets say you have a div which should be displayed when a user doesn't fill in an input field. We would use the @If directive like this. As you can see the property error is true, which means the element with the @If directive will be displayed.

The @If directive is used when you are wanting to render an element dependent on a boolean value in your data set. Like in the example above, we use @If to display user feedback if the user did not type in the required fields.

The @Bind directive is very useful. You can use this directive to bind a property in your View Object to an attribute on an element. For instance, if you want to apply a class to an element if an error occurs. You would use the @Bind:class attribute to do so.

Binding Type Description
@Bind:id String Allows you to bind your prop to the ID of the element
@Bind:class String Allows you to bind your prop to the CLASS of the element
@Bind:disabled Boolean Will add a disbaled attribute to the element depending on the value of your prop
@Bind:checked Boolean This will set a checked attribute to the element depending on the value of your prop
@Bind:href String Will set the href attribute to the value of your prop
The @On directive is used to bind a users action to an element. For example, if you have a button to alert a user's name once it's clicked. You would use the @On:Click directive and this would add an onclick event listener to that element.
Binding Type Description
@On:Click String This will add a click event to your element
@On:Enter String This enables you to add an enter event to your element
@On:Submit String This allows you to add a submit event to a form
